Control Structures in C

Conditional Statements


// If statement
if (age >= 18) {
    printf("You can vote!");
}

// If-else statement
if (score >= 60) {
    printf("You passed!");
} else {
    printf("You failed.");
}

// If-else if-else
if (grade >= 90) {
    printf("A");
} else if (grade >= 80) {
    printf("B");
} else if (grade >= 70) {
    printf("C");
} else {
    printf("F");
}

Conditional statements allow your program to make decisions:

  • Simple if statements for single conditions
  • if-else for two alternative paths
  • if-else if ladder for multiple conditions
  • Nested if statements for complex decision making

Switch Statement


switch (day) {
    case 1:
        printf("Monday");
        break;
    case 2:
        printf("Tuesday");
        break;
    case 3:
        printf("Wednesday");
        break;
    default:
        printf("Other day");
}

Switch statements are ideal for multiple choice scenarios:

  • More efficient than multiple if-else statements
  • Works with integers and characters
  • Requires break statements to prevent fall-through
  • Default case handles unmatched values

Loops


// For loop
for (int i = 0; i < 5; i++) {
    printf("%d ", i);
}

// While loop
int count = 0;
while (count < 5) {
    printf("%d ", count);
    count++;
}

// Do-while loop
int num = 0;
do {
    printf("Enter a number (0 to exit): ");
    scanf("%d", &num);
} while (num != 0);

Loops allow you to repeat code execution:

  • for loop: Used when number of iterations is known
  • while loop: Used when condition needs to be checked before execution
  • do-while loop: Used when loop body must execute at least once

Jump Statements


// Break statement
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    printf("%d ", i);
}

// Continue statement
for (int i = 0; i < 5; i++) {
    if (i == 2) continue;
    printf("%d ", i);
}

// Goto statement (use sparingly)
int i = 0;
start:
    if (i < 5) {
        printf("%d ", i);
        i++;
        goto start;
    }

Jump statements alter the flow of program execution:

  • break: Exits the current loop or switch
  • continue: Skips remaining loop body and continues with next iteration
  • goto: Jumps to a labeled statement (use with caution)
  • return: Exits the current function

Best Practices

  • Always use braces {} even for single-line blocks
  • Avoid deep nesting of control structures
  • Use switch instead of long if-else chains when possible
  • Avoid goto statements in modern code
  • Keep loop conditions simple and clear
  • Use meaningful variable names in loop counters
