Variables and Data Types in C

Basic Data Types


int age = 25;              // Integer type
float price = 99.99;       // Floating-point type
char grade = 'A';          // Character type
double pi = 3.14159265359; // Double precision floating-point

C provides several basic data types that are essential for programming:

  • int: Used for whole numbers (-2147483648 to 2147483647)
  • float: For decimal numbers (6 decimal places precision)
  • char: For single characters and small numbers
  • double: For high-precision decimal numbers

Modifiers


short int small_number = 32767;
long int big_number = 2147483647L;
unsigned int positive_only = 4294967295;

Type modifiers change the range and behavior of basic data types:

  • short: Reduces the size and range
  • long: Increases the size and range
  • unsigned: Removes negative numbers, doubling the positive range
  • signed: Allows negative numbers (default for most types)

Derived Data Types


int numbers[5] = {1, 2, 3, 4, 5};  // Array
struct Person {                     // Structure
    char name[50];
    int age;
    float height;
};

C also supports complex data types built from basic ones:

  • Arrays: Collections of similar data types
  • Structures: Groups of related data of different types
  • Pointers: Variables that store memory addresses
  • Unions: Special structures where members share memory

Type Conversion


int x = 10;
float y = (float)x;    // Explicit conversion
float z = 3.14;
int w = z;             // Implicit conversion

Learn about type conversion in C:

  • Implicit conversion (automatic type conversion)
  • Explicit conversion (type casting)
  • Type promotion in expressions
  • Potential data loss during conversion
